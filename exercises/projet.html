Indications

Urls

The webservice is hosted on Heroku at the adress http://parlezvous.herokuapp.com. You don't need to fill in the port as required during the formation. The root url is used in the browser to register a new user.

This is the urls list you will have to use:

/connect/{user}/{password}
Check the credentials user/password and returns the string true if the user is registered, false otherwise.
/message/{user}/{password}/{message}
Post a message with author user. The combination user/password must be valid for the message to be posted. No return value is sent by the server.
/messages/{user}/{password}
Receive the whole message list from the server. The messages are sent sorted chronologically. The list received uses the following syntax:
author1:message1;author2:message2; … authorN:messageN You will have to split the string to get all these informations. Use the methods provided by the String class.

For each request, if the combination user/password is erroneous, the request won't go through and you will receive the message: access denied

ListView

  • ListView is the component for displaying a list of elements.
  • A ListView requires an Adapter to work properly. The Adapter makes the connection between the raw data and the effectively displayed elements.
  • You have the possibility either to use default implementations of Adapters or create your own. This last solution allows you to create more advanced views and have better control over it.
